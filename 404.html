Samuel Downloads

Debian

Add the APT repository for Samuel applications: 

      echo "deb [trusted=yes] https://dl.samuelsharp.com debian/" > /etc/apt/sources.list
This is tested on Debian, but should also work on its deriviatives (i.e. Ubuntu, Kali).

Other Linux

Samuel software is compatible with any amd64 Linux machine.

If you don't have apt, direct downloads are available in the GitHub repo samuelbeep-dl/src.

Don't be fooled by the exe extension. Everything in the /linux folder is an ELF binary.

Windows

First, download the Scoop package manager: 

      Invoke-Expression (New-Object System.Net.WebClient).DownloadString('https://get.scoop.sh')
If that doesn't work, see scoop.sh.

Next, add the Samuel Scoop Bucket:

      scoop bucket add samuel https://github.com/samuelbeep-dl/windows

MacOS

We don't currently offer MacOS downloads of our software.
